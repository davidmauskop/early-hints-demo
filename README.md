# Early Hints Demo

This repositry can be shown [Early Hints](https://kazuho.github.io/early-hints/) Demo.
Early Hints can send pushable resources as hints. 

# Usage

```
$ npm install
$ npm start

$ npm run h2o
```

# This demo is using gophertiles demo.

https://http2.golang.org/

# Requirements

- Node.js
- h2o
